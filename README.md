# Zomato Data Analysis Using Python

## 📌 Project Overview
This project focuses on **analyzing Zomato restaurant data** using Python. The goal is to explore the dataset, clean and preprocess it, and generate meaningful insights into restaurant trends, ratings, costs, and cuisines.

## 📂 Dataset
The dataset contains various attributes of restaurants listed on Zomato, such as:
- Restaurant name
- Location
- Cuisine type
- Average cost for two
- Online ordering availability
- Ratings
- Votes

**Note:** Place the dataset file (e.g., `Zomato-data.csv`) in the project directory before running the notebook.

## 🛠 Technologies Used
- **Python 3**
- **Pandas** – Data manipulation
- **NumPy** – Numerical operations
- **Matplotlib & Seaborn** – Data visualization
- **Jupyter Notebook** – Interactive coding environment

## 📊 Key Steps in the Analysis
1. **Importing Libraries & Dataset**
2. **Data Cleaning**
   - Handling missing values
   - Renaming columns
   - Removing duplicates
3. **Exploratory Data Analysis (EDA)**
   - Distribution of ratings
   - Popular cuisines
   - Cost analysis
   - Location-based trends
4. **Visualization**
   - Bar plots, pie charts, histograms
   - Correlation heatmaps
5. **Insights & Conclusion**

## 🚀 How to Run
1. **Clone this repository**
   ```bash
   git clone https://github.com/yourusername/zomato-data-analysis.git
   ```
2. **Navigate into the project folder**
   ```bash
   cd zomato-data-analysis
   ```
3. **Install required libraries**
   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
4. **Open the Jupyter Notebook**
   ```bash
   jupyter notebook "Zomato Data Analysis Using PythonGH.ipynb"
   ```
5. **Run the cells** in order to reproduce the analysis.
